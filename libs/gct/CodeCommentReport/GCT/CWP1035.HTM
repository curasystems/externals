GCT.Util.TimeScheduler.IntTask.CompareTo Function
Compares two tasks with each other

Public int CompareTo (object)
  Type Name Description  
    object obj Object to compare  
  Return Description  
    int An integer representation of the comparison

  
Remarks:
If obj is not instance of IntTask, then return -1 If obj is instance of IntTask, compare the contained tasks' next execution times. If these times are equal, then order them randomly but consistently!: return the diff of their hashcode() values
