GCT.Protocols.STABLE.handleStableGossip Function
Called when a stability gossip is received.

Private void handleStableGossip (GCT.Address, GCT.Digest)
  Type Name Description  
    Address sender Sender of the gossip  
    Digest d The digest on the senders channel  
  Return Description  
    void    
Remarks:
Digest d contains (a) the highest seqnos deliverable for each sender and (b) the highest seqnos seen for each member. (Difference: with 1,2,4,5, the highest seqno seen is 5, whereas the highest seqno deliverable is 2). The minimum of all highest seqnos deliverable will be taken to send a stability message, which results in garbage collection of messages lower than the ones in the stability vector. The maximum of all seqnos will be taken to trigger possible retransmission of last missing seqno.
