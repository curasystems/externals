GCT.Blocks.PullPushAdapter.handleMessage Function
Called once a message is received from the Transport.

Protected void handleMessage (GCT.Message)
  Type Name Description  
    Message msg Message received from the Transport  
  Return Description  
    void    
Remarks:
Check whether the message has an identifier. If yes, lookup the MessageListener associated with the given identifier in the hashtable and dispatch to it. Otherwise just use the main (default) message listener
