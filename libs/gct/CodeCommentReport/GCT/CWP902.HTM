GCT.Stack.Retransmitter.Entry Class
The entry associated with an initial group of missing messages with contiguous sequence numbers and with all its subgroups.

Access: Private
Base Classes: Task
  Members Description  
    low Lowest sequence number in the Entry

  
    high Highest sequence number in the Entry

  
    list List of groups of sequence numbers

  
    initialSender Sender of the messages

  
    retCmd Method to call when retransmitting

  
    Entry Constructor

 
    remove Remove the given seqno

 
    run For each interval, call the retransmission callback command

 
Remarks:
Groups are stored in a list as long[2] arrays of the each group's bounds. For speed and convenience, the lowest and highest bounds of all the groups in this entry are also stored separately

For Example:

- initial group: [5-34] - msg 12 is acknowledged, now the groups are: [5-11], [13-34]
