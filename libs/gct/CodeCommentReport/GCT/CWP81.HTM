GCT.Channel Class
Abstract class for any implementation of a channel

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
Implemented interfaces: Transport
  Members Description  
    up_handler All events will be sent to this class once received.

 
    channel_listener Allows listening on the channel. The listener will be notified of channel closing, etc.

 
    Channel Constructor: Uses predefined stack.

 
    Channel Constructor: Stack and properties are specified

 
    connect Connects the Channel to a group.

 
    disconnect Disconnects the Channel from the group

 
    close Disconnects and closes the Channel.

 
    open Re-opens a closed channel.

 
    isOpen Checks if the Channel is Open.

 
    isConnected Checks if the Channel is Connected.

 
    send Sends a message through the Channel

 
    down Passes an event down the protocol stack

 
    receive Receives an event from the channel

 
    peek Performs the same as receive() but does not remove the Event

 
    getView Returns the current view of the Channel.

 
    getLocalAddress Returns the current Address (IP + Port) the Channel is using

 
    getChannelName Returns the name of the group the Channel is connected to.

 
    setUpHandler Sets the UpHandler that will receive all Events from the Channel

 
    setChannelListener Sets the ChannelListener that will be notified of changes to the channel

 
    setOpt Sets a variety of options within the channel
