GCT.GroupChannel Class
Channel for group communication (including .NET Remoting)

Author: Chris Koiak

Date: 12/03/2003

Access: Public
Base Classes: Channel
Implemented interfaces: IChannelSender, IChannelReceiver

Remarks:
This Channel can be used as a standard group communication channel or as a group extention to the .NET Remoting framework
