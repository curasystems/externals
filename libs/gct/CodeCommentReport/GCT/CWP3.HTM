GCT.Stack.AckReceiverWindow Class
Stores messages that have been received and acknowledged.

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
  Members Description  
    initial_seqno Sequence number of the first expected message

  
    next_to_remove Sequence number of the next message to be delivered

  
    msgs Collection of messages received

  
    AckReceiverWindow AckReceiverWindow Constructor

 
    add Adds a message

 
    remove Removes the next received message. Will only return a Message if all message before it have been received.

 
    reset Clears all received messages and resets the expectant sequence number.

 
    ToString Returns string representation of the object

 
Remarks:
Counterpart of AckSenderWindow. Every message received is ACK'ed (even duplicates) and added to a hashmap keyed by seqno. The next seqno to be received is stored in next_to_remove. When a message with a seqno less than next_to_remove is received, it will be discarded. The remove() method removes and returns a message whose seqno is equal to next_to_remove, or null if not found.
