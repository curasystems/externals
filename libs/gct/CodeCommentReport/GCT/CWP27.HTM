GCT.Stack.Protocol Class
Base class for all Protocol layers.

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
  Members Description  
    props Properties to be set

  
    up_protocol Protocol 'above' this Protocol

  
    down_protocol Protocol 'below' this Protocol

  
    stack ProtocolSinkStack this Protocol belongs to

  
    up_queue Queue of messages that have been passed up

  
    down_queue Queue of messages that have been passed down

  
    up_handler Thread for passing messages up in to the Protocol

  
    down_handler Thread for passing messages down in to the Protocol

  
    down_thread Determines whether the down_handler thread should be started

  
    up_thread Determines whether the up_handler thread should be started

  
    name Each Protocol must have a unique name

  
    init Initilises the Protocol

  
    start Starts the Protocol

  
    stop Stops the Protocol

  
    getUpQueue Returns the Up Queue

 
    getDownQueue Returns the Down Queue

 
    requiredUpServices List of events that are required to be answered by some layer above.

 
    requiredDownServices List of events that are required to be answered by some layer below.

 
    providedUpServices List of events that are provided to layers above (they will be handled when sent down from above).

 
    providedDownServices List of events that are provided to layers below (they will be handled when sent down from below).

 
    Name Gets and sets the unique Protocol name

 
    UpProtocol Gets and sets the Protocol above this Protocol

 
    DownProtocol Gets and sets the Protocol below this Protocol

 
    ProtocolSinkStack Sets the ProtocolSinkStack associated with this Protocol

 
    startUpHandler Starts the Up Handler thread if up_thread is true

 
    startDownHandler Starts the Down Handler thread if down_thread is true

 
    stopInternal Stops the Up and Down Handler threads

 
    receiveUpEvent Receive an event from the layer below

 
    receiveDownEvent Receive an event from the layer above

 
    passUp Causes the event to be forwarded to the next layer up in the hierarchy.

 
    passDown Causes the event to be forwarded to the next layer down in the hierarchy.

 
    up An event was received from the layer below.

 
    down An event is to be sent down the stack.

 
    setPropertiesInternal Removes and sets all generic properties of a Protocol, then passes remainder of properties on to implementation

 
    setProperties Sets the properties specified in the configuration string

 
    downHandler Thread to allow messages to be passed down asynchronously

 
    upHandler Thread to allow messages to be passed up asynchronously
