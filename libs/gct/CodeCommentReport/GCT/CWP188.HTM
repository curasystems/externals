GCT.Digest.merge Function
Merge member into Digest. Also called from merge(Digest).

Public void merge (GCT.Address, long, long, long)
  Type Name Description  
    Address sender Member to merge into Digest  
    long low_seqno Lowest sequence number associated with this member  
    long high_seqno Highest sequence number associated with this member  
    long high_seqno_seen Highest sequence number seen associated with this member  
  Return Description  
    void    
Other overloads:
GCT.Digest.merge(Digest)
Remarks:
Similar to add(), but if the sender already exists, its seqnos will be modified (no new entry) as follows: this.low_seqno=min(this.low_seqno, low_seqno) this.high_seqno=max(this.high_seqno, high_seqno) this.high_seqno_seen=max(this.high_seqno_seen, high_seqno_seen) If the sender doesn not exist, a new entry will be added (provided there is enough space)
