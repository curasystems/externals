GCT.Protocols.VectorTimeStamp Class
Vector Time Stamp for causal message ordering.

Access: Public
Base Classes: Object
  Members Description  
    members Sorted list of members. Key=Address Value=Number Messages sent

  
    owner Address of the owner of the timestamp

  
    ownerIndex Index of the owner inside the members collection

  
    VectorTimeStamp Constructor. Initialises the Vector time stamp to [0].

 
    Members Gets the current members in the vector time stamp

 
    Values Gets the current values in the vector time stamp

 
    getLocalTimeStamp Returns the number of messages sent by the owner

 
    increment Increments the owners count by 1

 
    getTransportableVectorTimeStamp Returns a representation of the object in a smaller transportable form.

 
    reset Sets all the values equal to zero.

 
    max Increases each value in the time stamp to the larger of either the current value or the corresponding value in the specified time stamp.

 
    causallyNext Calculates if the value is causally next to the current value.

 
    getValues    
    merge Any members not present already will be added to the time stamp vector.
