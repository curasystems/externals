GCT.Stack.Configurator Class
Sets up ProtocolSinkStack based on configuration string.

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
  Members Description  
    setupStack Sets up the ProtocolSinkStack with the specified configuration string.

 
    parseConfigurations Creates a ProtocolConfiguration instance for each protocol in the configuration string

 
    createProtocols Creates an array of Protocols from the configurations

 
    sanityCheck Checks all Protocols are unique and that each required up/down service is provided

 
    providesUpServices Check whether any of the protocols 'below' end_index provide Event (evt_type)

 
    providesDownServices Check whether any of the protocols 'above' end_index provide Event (evt_type)

 
    connectProtocols Connects all the Protocols together.

 
    getBottommostProtocol Moves down the stack until the last Protocol is discovered

 
    startProtocolStack Starts the Up and Down Handler threads in every Protocol.

 
    stopProtocolStack Stops all Protocol Up and Down Handler threads.

 
    ProtocolReq Internal Class for holding a Protocols required and provided services

 
    ProtocolConfiguration Internal Class that holds the Configuration for a Protocol

 
Remarks:
This class is responsible for creating instances of the protocol layers from the textual configuration string. Once the layers are created, they will be configured.
