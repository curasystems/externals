GCT.Membership Class
Used by the GMS to store the current members in the group

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
  Members Description  
    members List of current members

 
    Membership Constructor: Initialises with no initial members

 
    Membership Constructor: Initialises with the specified initial members

 
    getMembers Returns a clone of the members list

 
    setMembers Sets the members to the specified list

 
    add Adds a new member to this membership.

 
    add Adds a number of members to this membership

 
    remove Removes the specified member

 
    remove Removes a number of members from the membership

 
    clear Removes all members

 
    set Sets the membership to the members present in the list

 
    set Sets the membership to the specified membership

 
    contains Returns true if the provided member belongs to this membership

 
    copy Returns a copy of this membership.

 
    Clone Clones the membership

 
    size The number of members in the membership

 
    elementAt Gets a member at a specified index

 
    toString String representation of the Membership object

 
Remarks:
Coupled with an ArrayList, this class extends its facilites and adds extra constraints
