GCT.Stack.Retransmitter.RetransmitCommand.retransmit Function
Get the missing messages between sequence numbers first_seqno and last_seqno. This can either be done by sending a retransmit message to destination sender (nak-based scheme), or by retransmitting the missing message(s) to sender (ack-based scheme).

Public void retransmit (long, long, GCT.Address)
  Type Name Description  
    long first_seqno The sequence number of the first missing message  
    long last_seqno The sequence number of the last missing message  
    Address sender The destination of the member to which the retransmit request will be sent  
  Return Description  
    void
