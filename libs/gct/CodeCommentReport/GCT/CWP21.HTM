GCT.Util.MQueue Class
Wrapper class of Queue. Extends operations by allowing remove and peek operations to have timeouts

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
  Members Description  
    closed If MQueue is closed then no messages can be removed.

  
    q Internal Queue

  
    MQueue Constructor.

 
    Count Number of Objects in the queue

 
    Closed Gets and sets the access state of the queue

 
    Add Adds an object to the queue

 
    Remove Removes the first element in the queue.

 
    Remove Removes the first element in the queue. Waits indefinately for an element to arrive

 
    peek Returns the first element in the queue.

 
    close Closes access to the queue

 
    reset Closes the queue, clears all elements and then opens the queue for access
