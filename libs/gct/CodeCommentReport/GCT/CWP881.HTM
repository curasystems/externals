GCT.Stack.Retransmitter.RetransmitCommand Interface
Retransmit command used to retrieve missing messages

Access: Public
  Members Description  
    retransmit Get the missing messages between sequence numbers first_seqno and last_seqno. This can either be done by sending a retransmit message to destination sender (nak-based scheme), or by retransmitting the missing message(s) to sender (ack-based scheme).
