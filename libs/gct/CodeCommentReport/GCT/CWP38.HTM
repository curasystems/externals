GCT.Address Class
Identifes an endpoint. Contains an IPAddress and a port.

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Object
Implemented interfaces: IComparable
  Members Description  
    ipAddr    
    port    
    Address Constructor.

 
    Address Constructor.

 
    Address Constructor: will use localhost IP Address

 
    IP Get or Set the current IP Address

 
    Port Get or Set the current IP Port

 
    isMulticastAddress Checks whether the IP Address is a multicast address.

 
    Equals Checks for Equality with another object. Overrides base class method.

 
    GetHashCode Overrides the base class method. Used when the Address used as a key in Hashtables.

 
    CompareTo Compares the current object to the specified object. The object must be an Address. An Address is said to be larger if it's IP Address is greater, or the IP Address are the same but the port number is greater.

 
    ToString Returns string representation of the object

 
    Copy Returns a deep-copy of the Address

 
Remarks:
An Address is also used to uniquely identify a member in a group. Each machine can only have 1 locol port open for receiving unicast messages.
