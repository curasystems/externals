GCT.Protocols.DISCARD Class
Protocol: Discards up or down messages based on a percentage.

Author: Chris Koiak, Bela Ban

Date: 12/03/2003

Access: Public
Base Classes: Protocol
  Members Description  
    probUp probability of dropping up msgs

 
    probDown probability of dropping down msgs

 
    excludeItself if true don't discard messages sent/received in this stack

 
    localAddress Local Address

 
    random Used to randomly discard messages.

 
    DISCARD Constructor

 
    getName Returns unique Protocol name

 
    setProperties Sets the properties specified in the configuration string

 
    up Processes Events traveling up the stack

 
    down Processes Events traveling down the stack

 
Remarks:
This Protocol should only be used for testing
