Remarks

Counterpart of AckSenderWindow. Every message received is ACK'ed (even duplicates) and added to a hashmap keyed by seqno. The next seqno to be received is stored in 

next_to_remove
. When a message with a seqno less than next_to_remove is received, it will be discarded. The 
remove()
method removes and returns a message whose seqno is equal to next_to_remove, or null if not found.
