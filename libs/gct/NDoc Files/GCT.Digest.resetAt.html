Digest.resetAt Method 

Resets the seqnos for the sender at 'index' to 0.

public void resetAt(
   int index
);

Parameters

index
Index to reset

Remarks

This happens when a member has left the group, but it is still in the digest. Resetting its seqnos ensures that no-one will request a message retransmission from the dead member.
